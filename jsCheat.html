let The let keyword lets you create a variable that has Block Scope.
Like all variables, it can't be redeclared.
let x = "Help";
let x = 0; /* <-- Results in error */
function The function keyword defines a function. Keep in mind that in a loosely typed language such as JavaScript,
the functions don't state if they have a return value or not, so list it just in case.
function int_doSomething(int_x){
return 5 * int_x;
}
String.length String.length returns the current length of the string you're using.
let x = "Help";
console.log(x.length); /* <-- Results in 4 */
slice() String.slice() is a function to extract part of a string.
It takes up to 2 parameters (min 1.), extracts from the first index to the second, or if the second index isn't specified, it will cut the rest of the string.
let x = "Help";
let y = x. slice(1,3); /* <-- Returns el */
getElementById() Searches an HTML element by it's id attribute, and allows you to make changes to it.
document.getElementById("hello").innerHTML = "Help";
.innerHTML When retrieving an element using getElementById(), you can modify the internal HTML.
This can affect text. Observe:

This text will be modified


function example(){
let element = document.getElementById("p_modifyMe");

element.innerHTML = "This text has been modified";
}
.style Appending .style, as well as any style tag, to getElementById() can allow for changing the style.
The above example uses it to change the modified text to green.
document.getElementById("hello").style.color = "lime";
For Loop Standard For Loop. Exact same syntax as C++/Java:
/* for( initial variable; continue condition; update) */
for(let i = 0; i < 10; i++){
/*Code you want to run over and over again*/
}
While Loop A while loop, which keeps going so long as the condition in the center is true;
/* while (conidtion) */
while (condition){
/*Code you want to run over and over again*/
}
Math.random() Generates a random number.
let num = Math.random();
console.log(num); /* Output will be a random decimal */
=== === is the true equality operator. Where it measures if the two values are both the same type AND value.
let x = "5";

x === "5"; /* Outputs true */
x === 5; /* Outputs false */
console.log() Whether for debugging or just displaying crude output, console.log() can send it to the website's console.
use strict Enables the "strict mode" directive in JavaScript. With Strict mode, JavaScript becomes less loosely typed, good for keeping good code practice.
"use strict";
x = "5";
/* <-- Causes Error since variable wasn't declared */
Arrays Arrays can hold more than one value thanks to pointers, think of them as lists.
let int_arr = [1,2,3,4];
console.log(int_arr);
/* Outputs "1,2,3,4" */
BigInt For when you need a massive number, BigInt helps.
Javascript numbers allocate 8 bytes compared to the usual 4, so if you need something bigger than that, fear not.
let x = BigInt(1234567890123456789012345);
/* Just make sure not to mix up BigInts and Numbers, as there's data loss involved due to size differences */
