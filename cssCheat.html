KEY VALUE
. The class selector, primarily used as .example {}
Used when selecting HTML that have the class attribute filled. Observe:

/* HTML */

<div class="example">

/* INSERT HTML HERE */
</div>

/* CSS */
.example{
color: blue;
}
# The ID selector. Used for selecting specific unique elements with a unique ID that's mainly used for JavaScript. Less universal than the class selector but just as applicable.
/* HTML */

<div id="example">

/* INSERT HTML HERE */
</div>

/* CSS */
#example{
color: blue;
}
border The border key is short-hand when styling a border for an element.
border: 2px white dashed;
Generates this
ViewPort Values Want to style something that sizes itself with the size of your window? Use viewport width values from 1-100!

VW - Viewport Width.
VH - Viewport Height.
VMIN - Automatically uses the smallest Viewport value.
VMAX - Automatically uses the biggest Viewport value.
> The Child Selector > is used for when you want to select all elements that are under another element.
div > p {
width: 30vw;
}
:nth-child() :nth-child() is a very tricky selector. Remember, it's used for selecting a specific child index of that type.
div > a:nth-child(2){
/* Anything that's here will apply to every second anchor element inside any div */
}
:nth-of-type() :nth-of-type() Used for selecting a specific index of any element
div > a:nth-of-type(2){
/* Anything that's here will apply to every second anchor element */
}
var() A function used when referencing a variable, preferrably a global variable.
border: var(--AVariable);
calc() Used for solving math operations precisely. Can only use one operation per function.
width: calc(calc(100vw * 8) / 8);
Background The background property is a shorthand property, commonly for:
[color, image, repeat, position, origin]
body{
  background: blue url("background.gif") no-repeat fixed center;
}
margin Margin sets the space around the element.
div {
/*Sets the margin around the entire element to 10px*/
   margin: 10px;

/*Sets the margin above and below the element to 0, but uses auto to horizontally center it*/
   margin: 0 auto;
}
display Display sets how the element is treated. The default generally is either block or inline. The following are some of the more commonly used ones:
[inline, block, flex, inline-block]
img{
display: flex;
}
Float Float essentially moves an element to one side or the other,
moving other elements and texts out of the way within reason.
img{
float: left;
}
Clear Clear just moves an element below any objects that float to the direction clearance has been provided.
img{
float: left;
}

p{
clear: left;
}
Text-Align Text-Align just helps when centering certain things that are text based.
p{
text-align:: left | right | center | justify;
}
